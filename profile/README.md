# SignalWire Academy

Welcome to **SignalWire Academy** - the official training program for building AI-powered voice agents with the SignalWire AI Agents SDK.

## About

SignalWire Academy provides a structured, hands-on curriculum for developers learning to build conversational AI agents. Our courseware covers everything from basic agent creation to advanced enterprise deployments.

## Course Structure

### Level 1: Foundations
Build your first AI voice agent and learn core concepts.
- Hello World Agent
- First Live Call
- Order Lookup Functions

### Level 2: Intermediate
Master advanced features and integrations.
- Function Results & Actions
- DataMap Integration
- Skills System
- Context Management
- State Management
- Call Recording & Transfer
- Multi-Agent Systems
- Production Deployment
- Serverless Architecture

### Level 3: Advanced
Enterprise-grade patterns and best practices.
- Architecture Design
- Knowledge & RAG
- Performance Optimization
- Security & Compliance
- Monitoring & Observability
- Capstone Project
