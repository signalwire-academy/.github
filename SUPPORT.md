# Getting Help

## Course Support

### Students

If you're enrolled in SignalWire Academy and need help:

1. **Check the Course Materials**
   - Review the relevant lesson at [signalwire.academy](https://signalwire.academy)
   - Check the reference solutions in your assignment repo

2. **GitHub Issues**
   - For bugs in course materials, open an issue in the relevant repository
   - Include your assignment name, error messages, and steps to reproduce

3. **SignalWire Community**
   - Join discussions at [signalwire.community](https://signalwire.community/)
   - Search existing threads before posting

### Technical Issues

For issues with:
- **GitHub Classroom**: Check [GitHub's documentation](https://docs.github.com/en/education)
- **SignalWire Platform**: Visit [SignalWire Support](https://signalwire.com/support)
- **AI Agents SDK**: Check [SDK documentation](https://developer.signalwire.com/)
